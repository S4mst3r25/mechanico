## Installation
  
#### 1. Clone the repository
```
git clone https://github.com/S4mst3r25/mechanico/
```

#### 2.  Install dependencies
```
npm install
```

#### 3. Start development server
```
npm run dev
```

## Production

#### Build for production (output to /dist)
```
npm run build
```

#### Serve local production build
```
npm run preview
```
